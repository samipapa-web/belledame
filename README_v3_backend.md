# BELLE DAME COSMETIQUE — v3.1 (v3 + backend catalogue partagé)

## Lancer en local (1 PC)
### 1) Démarrer le backend
```bash
cd backend
npm install
npm start
```
Backend: http://localhost:8080

### 2) Ouvrir le site
Ouvrez http://localhost:8080 (le backend sert aussi le site)

## Utiliser depuis un autre téléphone sur le même Wi‑Fi
- Trouvez l’IP du PC (ex: 192.168.1.10)
- Ouvrez sur le téléphone : http://192.168.1.10:8080

## Synchroniser le catalogue
Dans le site : Gestion → renseignez **Admin PIN** → cliquez **Synchroniser DB**.
